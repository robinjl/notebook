## 快捷键

| 操作                   | 快捷键    |
| ---------------------- | --------- |
| 首页                   | ⌘ + ⇧ + H |
| 聚焦地址搜索框         | ⌘ + L     |
| 撤销上一次关闭的标签页 | ⌘ + ⇧+ T  |

| Momentum          | 新标签页背景图                |
| Octotree          | GitHub 文件树                 |
