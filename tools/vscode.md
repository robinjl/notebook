# VS Code

## Shortcuts

| 功能             | 快捷键                 |
| ---------------- | ---------------------- |
| 查看快捷键列表| ⌘ + K, ⌘ + S|
| 打开新窗口       | ⌘ + ⇧ + N    |
| 查看项目访问历史 | ⌘ + R            |
| 查看最近访问的文件| ⌘ + P |
| 定位到侧边栏     | ⌘ + 0            |
| 新建文件夹       | ⌘ + M（自定义）  |
| 新建文件         | ⌘ + N （自定义） |
